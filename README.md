# ToolCraft

## About

ToolCraft is a web platform that provides simple and useful online tools for working with files, text, images, and data.

## Features

- File conversion
- Data cleaning
- Text processing
- Image tools
- Calculators
- More tools coming soon

## Technologies

- HTML
- CSS
- JavaScript
- Django (planned)
- PostgreSQL (planned)

## Project Structure

The project is organized into separate folders for assets, styles, scripts, pages, and future backend components.
